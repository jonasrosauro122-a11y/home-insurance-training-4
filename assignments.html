Assignments and Activities

Each module includes an activity to help trainees apply their learning to real agency tasks. Below are the assignments for each course section:

  • Module 1: Identify examples of Coverage A–D using an actual homeowners policy.
  • Module 2: Complete a sample ACORD 80 form and label each section correctly.
  • Module 3: Simulate adding a new policy into an Agency Management System (AMS).
  • Module 4: Review a sample client file and identify possible data errors.

Agencies are encouraged to provide access to sample data, recorded AMS navigation, or policy examples for practice.
